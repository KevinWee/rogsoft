## Keywords

- [Clash](https://github.com/Dreamacro/clash) : A multi-platform & rule-base tunnel
- **[KoolClash](https://koolclash.js.org) : This project, a rule based custom proxy for [Koolshare Merlin] based on Clash.**

## Features

- HTTP/HTTPS and SOCKS protocol
- Surge like configuration
- GeoIP rule support
- Support Vmess/Shadowsocks/Socks5/Snell
- Support for Netfilter TCP redirect

## Installation

Download latest `koolshare.tar.gz` from [GitHub Release](https://github.com/SukkaW/Koolshare-Clash/releases), and then upload to Koolshare Merlin Soft Center as offline installation.

Read the [detailed installation instructions (written in Chinese)](https://koolclash.js.org/#/install) for more details.

## Clash on Other Platforms

- [Clash for Windows](https://github.com/Fndroid/clash_for_windows_pkg) : A Windows GUI based on Clash
- [clashX](https://github.com/yichengchen/clashX) : A rule based custom proxy with GUI for Mac base on clash
